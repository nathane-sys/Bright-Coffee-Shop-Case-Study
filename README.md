# Bright Coffee Shop Sales Analysis

## Project Description
This project analyzes transactional coffee shop sales data to uncover key business insights related to revenue, customer behavior, and product performance.

---

## Overview
The goal of this analysis is to transform raw data into actionable insights that support business decisions such as:
- What products drive the most revenue?
- When are peak sales periods?
- Which store performs best?

---

## Approach & Methodology

The project followed a structured data analysis workflow:

1. **Data Understanding**
   - Explored dataset structure
   - Checked date ranges and categories

2. **Data Validation**
   - Checked for null values
   - Identified duplicates
   - Validated numeric fields

3. **Data Transformation (SQL)**
   - Created calculated fields:
     - total_amount (Revenue)
     - Time buckets (30-minute intervals)
     - Time of day (Morning / Afternoon / Evening)
     - Day type (Weekday vs Weekend)
   - Extracted date features (month, year, day)

4. **Data Analysis (Excel)**
   - Built pivot tables
   - Explored trends and aggregations

5. **Data Visualization (Power BI)**
   - Built interactive dashboard
   - Designed KPI metrics and charts

---

## Key Insights

- ☀️ **Morning generates the highest revenue**
- ☕ **Coffee and Tea dominate product sales**
- 🏬 **Store performance is relatively balanced**
- 📈 **Revenue shows an upward trend over time**
- 📦 Some product categories contribute minimal revenue → potential optimization opportunity
